# Skills To Install in Cowork
Last Updated: April 2026

The following skills are written and ready. They exist as .skill files in your workspace folder but cannot be installed automatically — they need to be installed through the Cowork plugin interface.

## How to Install

Each .skill file is in: `itsmomapproved.com/[skill-name].skill`

In the Cowork app, go to Plugins or Skills settings and install each one from the workspace folder.
